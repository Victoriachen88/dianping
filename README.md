# 🍜 探店美化机 · 黄油风

> 一款移动端优先的**探店图片美化 + 文案生成**单页应用。上传美食探店照片，一键套用氛围滤镜、添加花字与贴纸、智能拼图，并自动生成小红书风格的探店文案（标题 + 正文 + 话题标签）。

零构建、单文件 `index.html`，双击即可运行；支持部署到 GitHub Pages 变成开放网址，手机（iOS / Android）也能正常使用。

---

## ✨ 功能特性

### 📤 图片上传
- 拖拽或点击上传多张图片，网格预览
- 底部「✨ 智能优化全部」一键批量标记

### 🎨 黄油相机风编辑
- **7 款氛围滤镜**：食欲大增 / 日系奶油 / 昭和胶片 / 清冷高级 / 原生鲜亮 / 复古拍立得 / 夜间闪光灯
- 滤镜**强度滑块**（0–100%），AI 按图片亮度/饱和度**自动推荐**
- **花字 A–F 六种样式**：描边立体 / 镂空线条 / 背景色块 / 手写飘带 / 双色叠加 / 弥散光底
- 3 种排版（居中大标题 / 角落标签 / 竖排文艺），内置智能文案语料
- **氛围贴纸托盘**：装饰 emoji + 探店功能贴纸（人均 / 定位 / 评分 / 排队）
- **8 组网感调色盘**，一键改花字渐变与描边
- 画布支持**拖拽、缩放、旋转、双击编辑**（基于 fabric.js，移动端触摸可用）
- 导出 **1080×1350（小红书竖图）/ 1:1**

### 🧩 智能拼图
- **📐 简单拼接**：2×2 网格 / 1×3 横排 / 左大右小 / 上下拼接 / Instagram 3+1 / 2×3 六宫格
- **✨ 创意拼图**：❤️ 心形 / 📰 杂志风 / 🎞️ 胶片风 / ⭕ 圆形组合 / 🎲 不规则拼贴
- 选图 → 选模板 → 一键生成 → 导出 PNG

### ⭐ 首图智能推荐
- 基于清晰度 / 饱和度 / 亮度采样评分，自动选出最佳首图，金边高亮
- 可点击其他图片替换首图

### ✍️ 探店文案生成
- 自动生成**吸睛标题 + 100–200 字正文 + 5–8 个话题标签**
- 三种文风：活泼 / 文艺 / 种草
- 支持**指定字数**（如「生成 150 字文案」）与**指定主题**（如「写火锅的文案」）
- 预留 OpenAI 接口：填入 API Key 走 AI 生成，否则回落本地模板

### 💬 对话式需求栏
底部输入框用自然语言即时改图改文，例如：
- `首图换成第 2 张`
- `滤镜全部改成冷色调`
- `文字改成好吃到哭`
- `文案语气再文艺一点`
- `请生成 100 字以上的内容`

## 🚀 快速开始

### 方式一：本地直接打开
下载后**双击 `index.html`**，用 Chrome / Edge / Safari 打开即可（需联网加载 CDN 资源）。

### 方式二：本地服务器
```bash
# Python 3
python -m http.server 8080
# 浏览器访问 http://localhost:8080/index.html
```

## 🌐 部署到 GitHub Pages（开放网址）

```bash
# 1. 在 GitHub 新建一个空仓库，然后：
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

推送后进入仓库 **Settings → Pages → Source 选 `main` 分支**，等 1–2 分钟即可获得开放网址：

```
https://你的用户名.github.io/仓库名
```

手机浏览器打开该网址即可使用。

## 🛠 技术栈

| 类别 | 技术 |
|---|---|
| 结构 | 原生 HTML / CSS / JavaScript（无框架、零构建） |
| 样式 | [Tailwind CSS](https://tailwindcss.com/)（Play CDN） |
| 画布编辑 | [fabric.js](http://fabricjs.com/) 5.3.0 |
| 拼图 / 导出 | 原生 Canvas API |
| 字体 | Google Fonts（ZCOOL KuaiLe / Ma Shan Zheng / Long Cang / Noto Sans SC 等） |
| 文案 | 本地模板碎片拼接 + 可选 OpenAI API |
